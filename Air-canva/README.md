Air Canvas Pro 🎨✋

Air Canvas Pro is a browser-based hand-tracking drawing app that lets you draw in the air using your fingers.
It uses MediaPipe Hand Landmarker to detect hand gestures in real time and turns them into drawing, erasing, and control actions.

---

✨ Features

- ✋ Real-time hand tracking with MediaPipe
- ☝ Draw with index finger
- ✌ Pause drawing
- 🖐 Erase using open palm
- 🎨 Color picker & brush size control
- 🌫 Opacity control
- ↩ Undo support
- 🧹 Clear canvas
- 🖥 Fullscreen immersive mode
- 🔍 Hand skeleton visualization
- ⚡ Smooth cursor interpolation for stable drawing

---

🧠 Gestures

Gesture| Action
☝ Index finger up| Draw
✌ Index + middle| Pause
🖐 Open hand| Erase

---

🚀 How to Run

1. Clone or download this repo
2. Open index.html in a modern browser (Chrome recommended)
3. Allow camera access
4. Start drawing in the air 🎉

No build step required — it runs entirely in the browser.

---

🛠 Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- MediaPipe Tasks Vision (Hand Landmarker)
- Web APIs (getUserMedia, Fullscreen API)

---

📌 Notes

- Works best in good lighting conditions
- Performance may vary depending on device GPU
- Only one hand is tracked at a time
